# Cyber-Attacks Targeting Smart EV Infrastructure

This repository contains the experimental results for a real-time deep-learning-based framework for detecting and mitigating measurement-layer cyber-attacks in AC electric vehicle charging infrastructure.

## Interactive Visual Results

The full interactive Plotly HTML reports are available in the **GitHub Releases** section because the files are too large to store directly inside the repository.

👉 **Download the visual results here:**  
[Interactive Plotly Visual Results](https://github.com/Usame-Sakkar/Cyber-Attacks-Targeting-Smart-EV-Insfrastructure/releases/download/visual-results-v1.0/Interactive_results.zip)

The release contains ZIP files with the interactive visual reports for the evaluated charging sessions and attack scenarios. These reports show the original measurements, attacked measurements, detector decisions, and mitigated outputs.


## How to Use the Visual Reports

1. Open the release link above.
2. Download the ZIP file containing the visual reports.
3. Extract the ZIP file.
4. Open the `.html` files in a web browser.
